# fastpost2026

مشروع ماركيت كوردية متعدد الأقسام (چێشتخانه، عمرە، سەیارەکان، ماركیت، إلكترونیات)

## للتشغيل محلياً:
```bash
npm install
npm run dev
```
- ثم افتح: http://daim-post.online

---

## ميزات المشروع الأساسية:
- Next.js 14 & TypeScript
- ملفات manifest و service worker لدعم PWA
- صفحة رئيسية + صفحة داشبورد بسيطة
- جاهز للنشر على Vercel أو Netlify بسهولة
